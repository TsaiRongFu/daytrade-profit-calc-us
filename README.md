# 📈 美股損益平衡計算器（Day Trade Profit Calculator – US）

一個 **簡單、直覺、免登入** 的美股當沖／短線交易  
👉 **損益平衡價計算工具**

幫助你快速計算：
- 💰 **買入 / 賣出後的損益**
- ⚖️ **回本所需的賣出價格**
- 🧾 **考量交易手續費後的實際結果**

## ✨ 功能特色

- ✅ 美股交易專用（USD）
- ✅ 支援手續費輸入（%）
- ✅ 即時計算損益 / 回本價
- ✅ 手機、桌機皆可使用
- ✅ 純前端，GitHub Pages 部署

## 🧮 適合誰使用？

- 📊 美股當沖 / 波段交易者  
- 🧠 想快速確認「**賣多少才不賠**」
